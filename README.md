# $ man bomber
A bomberman clone

## Building
### Requirements
* CMake >= 2.6

### Building
```
cmake ./
make
```
Binaries will be located in `bin/`.

### Running
* Server: manbomber-srv <ip address> <port>
* Client: manbomber-cli <ip address> <port> <player id (0 to 3 inclusive)>
